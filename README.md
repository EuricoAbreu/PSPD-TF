# Projeto Final - PSPD

## Descrição

Este projeto tem como objetivo explorar e avaliar a performance de sistemas paralelos e distribuídos utilizando **Kubernetes**, **Apache Spark** e **MPI/OMP**. A iniciativa visa testar a capacidade de **computação paralela e distribuída**, verificando aspectos como **escalabilidade, tolerância a falhas e orquestração de cargas de trabalho**.

## Objetivos

Os principais objetivos do projeto incluem:

- **Testar o desempenho e a tolerância a falhas do Kubernetes**
- Configuração de um cluster Kubernetes com um nó mestre e pelo menos dois nós trabalhadores.
- Monitoramento do cluster e análise de desempenho em diferentes cenários.
- Avaliação da resiliência da aplicação sob falhas controladas.

- **Integração do Apache Spark com Kubernetes**
- Implementação de um ambiente distribuído para processamento de grandes volumes de dados.
- Testes de carga e análise de benefícios da integração Spark-Kubernetes.

- **Execução de aplicações MPI/OMP sobre Kubernetes**
- Integração de aplicações paralelas utilizando MPI/OMP no cluster Kubernetes.
- Comparação do desempenho com os cenários anteriores.

## Metodologia

1. Configuração do cluster Kubernetes em um ambiente controlado.
2. Implementação de aplicações para testar computação distribuída.
3. Execução de testes para avaliar tempo de resposta, escalabilidade e resiliência.
4. Documentação dos resultados e análise comparativa entre as abordagens.

## Tecnologias Utilizadas

- **Kubernetes**
- **Apache Spark**
- **MPI / OpenMP**
- **Monitoramento via interface web e console**

## Atividade Extra (Opcional)

Além dos testes técnicos, também iremos explorar conceitos **Cloud Native**, abordando temas como:

- Benefícios do Kubernetes para DevOps.
- Comparação entre aplicações monolíticas e microsserviços.
- Monitoramento e observabilidade de aplicações distribuídas.
